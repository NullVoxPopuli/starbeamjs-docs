                          # export const LIFETIME

                          LIFETIME provides a mechanism for structured cleanup of objects.

                          Info

                          Structured cleanup means linking the cleanup of an object to the lifetime of its owner, and the ability to register cleanup handlers for objects.

                          # Events

                          # on.cleanup

                          Register a cleanup handler for an object. This function is called when the object is finalized.

                          # enqueueAction

                          Enqueue a function to be executed asynchronously in the action phase. The action phase is the first phase of rendering. Code that runs in the action phase is allowed free access to reactive values (both reads and writes).

                          function enqueueAction(...notifications): void;
                          notifications
                          Array<() => void>The notifications to enqueue.

                          # enqueueRender

                          Enqueue a function to be executed asynchronously in the render phase. The render phase is the second phase of rendering. Code that runs in the action phase is allowed to read reactive values, but may not write to them.

                          function enqueueRender(...notifications): void;
                          notifications
                          Array<() => void>The notifications to enqueue.

                          # enqueueAfterRender

                          Enqueue a function to be executed asynchronously after the render phase. This code runs after all render functions have finished.

                          function enqueueAfterRender(...notifications): void;
                          notifications
                          Array<() => void>The notifications to enqueue.

                          Multi-Phase Rendering

                          You can use afterRender to implement a multi-phase rendering algorithm, but you should be careful to make sure to avoid creating infinite loops.

                          This API is a power tool, and you should make sure there's not a one-pass way to accomplish your goal before resorting to it.
